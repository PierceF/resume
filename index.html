About

Anonymous Chat is a peer2peer chat application build during the Holochain Hackathon in Barcelona (15-18.11.2019). It serves as a simple proof of concept. Holochain is a framework to write decentralized applications, and allows nodes (users) to host the data themselves, getting rid of serverfarms and companies in between. It is a powerfull new design pattern to enable self-managing communities, which could have a huge impact on how humans interact with each others.

Tech

  • Holochain
  • Rust
